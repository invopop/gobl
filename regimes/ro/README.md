## Supplier Tax ID (CUI)

The CUI (Cod Unic de ÃŽnregistrare) is the Romanian company registration/tax number issued by ANAF:

- **Format:** 2â€“10 digits (numeric only).
- **Checksum:** Validated using the Romanian control digit algorithm with key `753217532`.
- **Normalization:** The `RO` country prefix is stripped automatically; spaces and non-numeric characters are removed.
- **Examples:** `13547272` (valid), `10864098` (valid).

## E-Invoicing

Since **1 July 2024**, B2B e-invoicing is mandatory in Romania for all transactions between VAT-registered entities. Invoices must be submitted via the national **RO e-Factura** platform operated by ANAF, in XML format (UBL 2.1 or CII) compliant with the **RO_CIUS** profile (an EN 16931-1 Core Invoice Usage Specification).

Support for RO_CIUS is tracked in the TODO section below. The full list of validation rules is in the [RO_CIUS Rules](#ro_cius-rules) section.
